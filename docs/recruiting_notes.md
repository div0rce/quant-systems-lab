# Recruiting Notes

> Internal notes on how to position this project.

## What this project demonstrates

1. Deterministic state machine design
2. Integer price modeling (no floating-point prices)
3. Binary protocol with explicit serialization
4. Replay/recovery from append-only logs
5. Invariant testing beyond happy-path demos
6. Reproducible benchmark methodology
7. Clean incremental engineering process

## What this project is NOT

- Not a production exchange
- Not a trading bot
- Not connected to real markets
- Not making profitability claims
- Not claiming "production-grade" or "battle-tested" status
